# Iris Flower Classification GUI

A Machine Learning GUI application built using Python, Tkinter, and Scikit-learn.

## Features
- Predicts Iris flower species
- Uses Decision Tree Classifier
- Simple GUI using Tkinter
- Saves trained model using Joblib

## Technologies Used
- Python
- Scikit-learn
- Tkinter
- Joblib

## Installation

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
python iris_gui.py
```

## Dataset
Uses the built-in Iris dataset from Scikit-learn.
